Problemet med att prata om siare/väktare är att det är ett sätt att spana efter reaktioner från folk. Inte att någon erkänner rakt av, men att man kan kolla efter vilka som reagerar på vad man säger. Ett varulvsknep är exempelvis att säga något i stil med "eftersom vi bara var nio spelare från början så tror jag vi kan ta för givet att det inte finns någon väktare bland oss, så om vi massclaimar och någon påstår sig vara väktare så skulle det definitivt göra mig misstänksam" och sedan hålla upp ögonen efter dem som säger "nä, vadå? så kan man inte säga" eftersom man kan räkna med att om det finns någon väktare så kommer denna att reagera på vad man skrivit.

Vi borgare bör alltså låta bli att gå med håven efter eventuella powerroller.
